# Source-Code
```
Terragreen Core-Wallet
```
## Prerequisite
|Modules | Version|
|--------|--------|
|[Node](https://nodejs.org/en/)|12.7.2 or above|
| [npm](https://docs.npmjs.com/about-npm-versions) |6.11.2 or above|



## Build
```
npm link
```
After linking, run following 
```
terragreen <command>
```
for help in commands type following in cmd
```
terragreen --help
```
